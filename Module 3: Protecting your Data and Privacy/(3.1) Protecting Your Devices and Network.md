# (3.1) Protecting Your Devices and Network

<u>Protecting Your Computing Devices:</u>
- Turn firewall on (and constantly update it)
- Install antivirus and antispyware (and constantly update it)
- Manage operating system and browser (and constantly update it)
- Set up password protection!

<u>Wireless Network Security at Home:</u>
- Wireless networks allow Wi-Fi enabled devices to connect to the network by way of a preset network identifier AKA **Service Set Identifier (SSID)**. 
- Hackers are aware of preset SSID and default password → these details should be changed → enable **WPA2 encryption** (scrambles your internet traffic), but still vulnerable
- **KRACKs** vulnerability = attackers break encryption between a wireless router and a wireless device, giving them access to network data.
	- Mitigate KRACKs:
		- update all wireless capable devices
		- use wired connection for any devices with wired **Network Interface Card (NIC)**
		- use trusted **Virtual Private Network (VPN)** service when accessing a wireless network

<u>Public Wi-Fi Risks:</u>
- Outside home, there are public wireless networks or Wi-Fi hotspots → risks
- You should:
	- verify that your device is NOT configured with file/media sharing and/or has files that require user authentication with encryption.
	- use encrypted VPN service to prevent **"eavesdropping"** (intercepting info)
	- turn of Bluetooth too when not in use

<u>Password Security & Guidelines:</u>
- Important that all your online accounts have a unique password. 
	- You can use a **password manager**!
- Strong Passwords:
	1. Do NOT use dictionary words or names 
	2. Do NOT use common misspellings of dictionary words
	3. Use special characters ! @ # $ % ^ & * ( ) 
	4. Do NOT use computer names or account names
	5. Use password with more than 10 characters
- **United States National Institute of Standards and Technology (NIST)** password requirements:
	- At least 8 characters
	- NO common passwords
	- NO composition rules
	- NO hints
	- etc.
- EXAMPLES:
	- Weak:
		- **Password123**
		- **Iloveyou**
		- **1q2w3e4r**
		- **April012000**
	- Strong:
		- **Am@z0nPa55**
		- **J0n@than#81**
		- **!Lik3MySch001**
