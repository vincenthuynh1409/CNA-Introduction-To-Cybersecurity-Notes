# (3.2) Data Maintenance

**Encryption** = process of converting information into a form in which unauthorized parties cannot read it → only the trusted, authorized person with the secret key or password can decrypt the data and access it in its original form.

<u>How to encrypt your data:</u>
- **Encrypting File System (EFS)** = Windows feature that can encrypt data
- Tutorial:
	- select one or more files or folders
	- right click selected data 
	- Select 'Properties' > 'Advanced'
	- Select 'Encrypt contents to secure data' check box
	- EFS encrypted! 

<u>Backup Your Data:</u>
- Home network (storing data locally)
- Secondary location (copy data to a **Network Attached Storage (NAS)**, external hard drive or thumb drives, CDs, DVDs, tapes, etc)
- The cloud (**Amazon Web Services (AWS)**, etc → access to account = access data)

<u>Deleted Data:</u>
- When you move a file to the recycle bin and delete it permanently, the file is only inaccessible from the operating system → anyone with the right forensics tools can still recover the file due to magnetic trace left in hard drive
- To delete data permanently:
	- must be overwritten by ones and zeros multiple times (tools such as **SDelete** for Microsoft Windows)
	- physically destroy the hard drive or storage device
