# (2.2) Methods of Infiltration

**Social Engineering** = manipulation of people into performing actions or revealing personal information/data → basically lying.
- *Pretexting* = attacker calls person and lies to gain access to privileged data.
- *Tailgating* = attacker quickly follows an authorized person into a secure, physical location secretly without consent
- *Something for something (quid pro quo)* = attacker requests personal information in exchange for something like a gift.

**Denial-of-Service (DoS)** = simple type of attack resulting in the interruption of network services to users, devices, or applications. 
- *Overwhelming quantity of traffic* = when network/app/host is sent an enormous amount of data at a rate which it cannot handle → slowdown and crashes
- *Maliciously formatted packets* = maliciously formatted *packet* (collection of data flowing between source and receiver over a network) sent and receiver will be unable to handle it → crashes and slowdowns

**Distributed Denial-of-Service (DDoS)** = DoS attack but originates from multiple, coordinated sources.

> - An attacker builds a network (*botnet*) of infected hosts called "zombies", which are controlled by handler systems.
> - The zombie computers will constantly scan and infect more hosts, creating more and more zombies 
> - When ready, the hacker will instruct the handler systems to make the botnet of zombies carry out a DDoS attack

**Botnet** = group of bot computers (infected), connected through the internet, that can be controlled by a malicious individual/group → activated to distribute malware, launch DDoS attacks, distribute spam email, or execute brute-force password attacks.

> - Infected bots try to communicate with a command and control host on the internet
> - Cisco Firewall botnet filter is a feature that detects traffic coming from devices infected with the malicious botnet code.
> - cloud-based Cisco Security Intelligence Operations (SIO) service pushes down updated filters to the firewall that match traffic from new known botnets.
> - Alerts go out to Cisco's internal security team to notify them about infected devices that are generating malicious traffic so they can prevent, mitigate, and remedy these.

**On-Path Attacks** = attackers intercept or modify communications between two devices to collect info or impersonate devices
- **Man-In-The-Middle (MITM)** = cybercriminal takes control of device without user's knowledge → intercepting/capturing user info/data before its sent to intended destination.
- **Man-In-The-Mobile (MITMO)** = attack used to take over mobile device → when infected, the mobile device is instructed to exfiltrate sensitive info and send it to attackers.

**SEO Poisoning** = attackers take advantage of popular search terms and SEO (Search Engine Optimization → gaining greater visibility in search engine results) to push malicious sites higher up the ranks of search results.

**Wi-Fi Password Cracking**:
- *Social Engineering* (EX: colleague asking for private wifi password)
- _Brute-Force attacks_ = testing possible password combinations and guessing
- _Network Sniffing_ = capturing packets sent on the network, and if password encrypted, they can still use a password cracking tool

**Password Attacks:**
- **Password Spraying** = "spraying" commonly used passwords across large number of accounts (EX: attacker uses 'password123' w/ many usernames before trying again with a second commonly use password like 'qwerty')
- **Dictionary Attacks** = attacker systematically tries every word in a dictionary or list of commonly used words as a password in attempt to break into an account.
- **Brute-Force Attacks** = simplest and most common way, by using all possible combinations of letters/numbers/symbols until they get it right.
- **Rainbow Attacks** = compares *hash* (numerical values uniquely identifying data like passwords) of a password with those stored in the *rainbow table* (large dictionary of precomputed hashes and passwords calculated) and when found match, they find the password used to create the hash.
- **Traffic Interception** = plain text or unencrypted passwords can easily be read by other humans/machines by interception communications.

**Advanced Persistent Threats (APTs)** = multi-phase, long-term, stealthy, and advanced operation against a specific target → typically well-funded and targets organizations/nations for business/political reasons.

<u>Method Matching:</u>

1) *On your way into the office, a person whom you have never seen before asks you to hold the door — they forgot their access card*

- ANSWER: **Social Engineering**

> Asking hold the door (needing an access card) is a social engineering attempt due to them manipulating you into getting access to an employee-only office.

1) *You have started getting an error message when accessing your computer: ‘Your connection was interrupted. A network change was detected.*

- ANSWER: **DoS**

> "Connection interrupted" and "network change" typically means that there is a network connection slowdown or crash probably due to an attacker using "overwhelming traffic" or "maliciously formatted packets" to interrupt the system.

1) *You searched for @Apollo’s website on Google, but when you clicked on the top result, you were redirected to a page advertising antivirus software*

- ANSWER: **SEO Poisoning**

> SEO poisoning manipulates search engine results so top results can be malicious pages/websites, and clicking on the top result directing you to a suspicious ad software means that it is SEO poisoning.
