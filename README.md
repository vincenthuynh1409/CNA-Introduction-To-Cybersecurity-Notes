# Cisco Networking Academy: Introduction to Cybersecurity Notes

## Description:

### 👋 Introduction:

This online introductory course will take me inside the world of cybersecurity. I will learn cybersecurity basics to protect my personal digital life and gain insights into the biggest security challenges companies, governments, and educational institutions face today. 

**Curriculum:**
- Module 1: Introduction to Cybersecurity
- Module 2: Attacks, Concepts and Techniques
- Module 3: Protecting your Data and Privacy
- Module 4: Protecting the Organization
- Module 5: Will Your Future Be in Cybersecurity?
- End of Course Survey
- Introduction to Cybersecurity: Course Final Exam

### 🗺️ Navigation:

There are 5 modules in the course, and there will be 5 folders that correspond to each module. To find the nootes, click on the specific module folder in the repository and navigate through the markdown files!
